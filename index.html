Who am I?

I'm a full stack web developer who enjoys programming, art, and graphic design. I started learning to code in 2015 using Scratch. I mostly created games and engines in Scratch, before learning other no-code app development tools such as MIT App Inventor. Now, I can program in JS, Python, R. as well as other languages. I have a passion for supporting Open Source projects and tools.
